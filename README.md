# UNO Web Game

A real-time multiplayer UNO web application featuring automated bot players and live chat. Built using Node.js, Express, PostgreSQL, and TypeScript.

## Features

- Multiplayer turn-based UNO gameplay
- Automated bot players
- Live in-game chat
- Session-based gameplay
- Dynamic UI updates
- Browser-based play experience

## Technologies

- Node.js
- Express
- PostgreSQL
- TypeScript
- HTML / CSS / JavaScript
